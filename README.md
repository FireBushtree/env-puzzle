## 📦 安装

```
yarn add env-puzzle
```

or

```
npm i env-puzzle
```

## 🔨 示例

```typescript
import {Template, CreateModal, ViewModal} from 'env-puzzle';

const App = () => (
  <>
    <Template></Template>
    <CreateModal />
    <ViewModal />
  </>
)

```

## ⌨️ 本地开发

克隆到本地开发:
```bash
$ git clone git@github.com:FireBushtree/env-puzzle.git
$ cd env-puzzle
$ npm install
$ npm start
```
